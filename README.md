 # Friday

 Friday 的个人开源工具展示站，基于 Next.js + Tailwind CSS + shadcn/ui 构建。

 ## 项目说明

 站点展示了我的开源项目、CLI 工具、浏览器自动化脚本和 Codex / AI Skill。

 项目数据位于 [`data/projects.json`](./data/projects.json)。

 ## 本地开发

 ```bash
 npm install
 npm run dev
 ```

 ## 构建

 ```bash
 npm run build
 ```

 构建产物输出到 `dist/` 目录。

## 部署

本项目配置为静态导出，可部署到 Vercel、Cloudflare Pages、GitHub Pages 等任意静态托管平台。

线上地址：https://friday-portfolio-fridays-projects-77e733de.vercel.app
